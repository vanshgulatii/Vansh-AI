# Vansh AI

A simple AI student assistant built with **Next.js** (frontend) and **FastAPI** (backend). It uses the OpenAI API for answering questions and SQLite with SQLAlchemy for data storage.

## Prerequisites

- Docker & Docker Compose installed
- An OpenAI API key (set in `backend/.env.example` and copied to `backend/.env`)

## Getting started (Docker)

```bash
# 1. Copy the env example and add your OpenAI key
cp backend/.env.example backend/.env
# Edit backend/.env and set OPENAI_API_KEY=your_key

# 2. Build and start the containers
docker compose up --build
```

- The FastAPI backend will be available at `http://localhost:8000`.
- The Next.js frontend will be available at `http://localhost:3000`.

## Local development (without Docker)

### Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
# Copy .env.example -> .env and set your key
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:3000` in a browser and ask a question.

## Database migrations

Alembic is set up for schema migrations.

```bash
# Create a new migration after changing models
alembic revision --autogenerate -m "description"
# Apply migrations
alembic upgrade head
```
